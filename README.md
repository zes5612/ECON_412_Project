# ECON_412_Project
Analysis of S&amp;P 500 historical data for RIT's ECON-412 Data Wrangling & Exploration Class

When running on your machine, make sure to set up a virtual environment with:
yfinance,
polars,
matplotlib

You simply need to run 'entire_process.py' to generate all of our CSV files and graphs.
Note: due to the number and size of the CSV files, they are hidden in this repository, but you
will be able to view them once you run the code on your machine.
